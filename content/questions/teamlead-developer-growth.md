## Короткий ответ

1:1 meetings, code review, pairing, learning sessions. Важно: давать сложные задачи, но с поддержкой.

## Контекст

Интервьюер хочет понять, как вы помогаетеjunior-ам расти.

## Как строить ответ

### Инструменты

1:1, code review, pairing, learning sessions.

### Поддержка

Как поддерживаете: mentoring, feedback, resources.

### Рост

Как трекаетеprogress: goals, milestones.

## Пример ответа

Инструменты: 1:1 weekly, code review, pairing, learning sessions. Поддержка: mentoring, feedback, books, courses. Рост: goals (3-6 месяцев), milestones. Результат: 2junior-а стали mid-level за год.

## Частые ошибки

- Не даватьfeedback
- Не трекатьprogress
- Давать слишком сложные задачи без поддержки
- Не мотивировать
