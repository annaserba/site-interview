## Короткий ответ

Funnel analysis: визуализация drop-off на каждом шаге. Этапы: define steps → collect data → identify drops → experiment → measure. Ключевое: найти biggest drop и оптимизировать.

## Контекст

Ключевой инструмент для conversion optimization. Проверяют понимание funnel methodology.

## Как строить ответ

### Этапы

1. Define steps: registration → onboarding → first action → purchase. 2. Collect: track events. 3. Identify: biggest drop. 4. Experiment: test solutions. 5. Measure: conversion improvement.

### Инструменты

Amplitude, Mixpanel, GA4, Hotjar (heatmap).

### Оптимизация

Simplify steps. Reduce friction. Add social proof. A/B test changes.

## Пример ответа

Funnel: 1000 visitors → 200 signup (20%) → 100 onboarding (50%) → 60 first action (60%) → 30 purchase (50%). Biggest drop: onboarding. Experiment: simplified form → +20% onboarding completion.

## Частые ошибки

- Не определять steps
- Не сегментировать funnel
- Игнорировать biggest drop
- Не проводить experiments
