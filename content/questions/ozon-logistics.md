## Короткий ответ

Ozon logistics: warehouse management, route optimization, demand forecasting, real-time tracking. Ключевые: ML для прогнозов, оптимизация маршрутов (vehicle routing), автоматизация складов.

## Контекст

Ключевой domain Ozon. Проверяют понимание logistics optimization.

## Как строить ответ

### Warehouse management

Inventory optimization. Pick/pack routing. Automation: robots, conveyors.

### Route optimization

Vehicle routing problem (VRP). ML для forecasting delivery time. Real-time traffic.

### Demand forecasting

ML models: sales history, seasonality, promotions. Inventory placement.

## Пример ответа

Route optimization: 1000 заказов → ML оптимизирует маршруты → 20% less driving time. Warehouse: robots moving shelves → picker path optimized. Forecasting: predicts demand → pre-positions inventory.

## Частые ошибки

- Не учитывать traffic conditions
- Игнорировать seasonal patterns
- Не оптимизировать warehouse layout
- Не мониторить delivery times
