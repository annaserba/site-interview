## Короткий ответ

p-value — вероятность при истинной нулевой гипотезе получить наблюдаемую статистику или более экстремальную согласно заранее выбранному критерию. Это не вероятность истинности H0 и не размер эффекта. Значение корректно только при выполнении допущений теста и stopping rule; его нужно читать вместе с effect size, доверительным интервалом, power и ценой ошибок.

## Контекст

Интервьюер проверяет корректную условную вероятность и способность не сводить решение к порогу 0.05.

## Как строить ответ

### Назвать условие

Вероятность считается при H0, а не для H0 после наблюдения данных.

### Связать с процедурой

Выбор статистики, хвостов, момента остановки и поправки на множественные тесты определяет валидность p-value.

### Перейти к решению

Оцените величину и неопределённость эффекта и сравните с минимально полезным изменением.


## Код из интервью

```python
from scipy import stats
import numpy as np

# A/B тест: t-test
control = np.random.normal(100, 15, 1000)
treatment = np.random.normal(105, 15, 1000)

stat, p_value = stats.ttest_ind(control, treatment)
print(f"t-statistic: {stat:.3f}, p-value: {p_value:.4f}")

# Доверительный интервал разницы
diff = treatment.mean() - control.mean()
se = np.sqrt(control.var()/len(control) + treatment.var()/len(treatment))
ci_low, ci_high = diff - 1.96*se, diff + 1.96*se
print(f"95% CI: [{ci_low:.2f}, {ci_high:.2f}]")
```

## Пример ответа

p-value — это вероятность получить наблюдаемый эффект или более экстремальный при условии, что нулевая гипотеза верна. Например, p-value = 0.03 означает: если бы H0 была верна, вероятность получить такой результат составила бы 3%. Это НЕ вероятность того, что H0 верна. На практике: p-value < 0.05 не означает практическую значимость — разница в 0.1% может быть статистически значимой, но бесполезной. Поэтому всегда смотрим на effect size и confidence interval. Также p-value зависит от размера выборки — при n=1000000 даже крошечная разница будет «значимой».

## Частые ошибки

- Говорить «вероятность случайности результата».
- Приравнивать статистическую значимость к практической.
- Игнорировать множественные сравнения.
