## Короткий ответ

Resume screening → phone screen → technical → system design → cultural fit → offer. Важно: оценивать не только скиллы, но и potential, culture.

## Контекст

Проверяют понимание процесса hiring и оценки кандидатов.

## Как строить ответ

### Этапы

Resume → phone → technical → design → culture → offer.

### Оценка

Что проверяете: coding, system design, communication.

### Решение

Как принимаете решение: feedback, scoring.

## Пример ответа

Этапы: Resume screening (10 минут) → phone screen (30 минут) → technical (1 час) → system design (1 час) → cultural fit (30 минут) → offer. Оценка: coding (40%), design (30%), culture (30%). Решение: feedback panel, scoring. Результат: hiring rate 20%, retention 90%.

## Частые ошибки

- Оценивать только скиллы
- Не проверять cultural fit
- Быстро нанимать под давлением
- Не давать feedback
